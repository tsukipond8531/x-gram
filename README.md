### :hammer: Installation

Firstly, install dependencies using yarn

```bash
git clone https://github.com/iamvucms/x-gram.git
cd x-gram
yarn
npx pod-install
```

### :car: Usage

Start the application

```bash
npx react-native run-ios # for ios
npx react-native run-android # for android

```
